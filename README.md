# QualiCheck:
QualiCheck é um projeto desenvolvido em sala de aula, na disciplina "Projeto Integrador I", ministrada pelo professor Walner Pessoa, durante o curso de Análise e Desenvolvimento de Sistemas, no CEUB. a fim de avaliar atendimentos na área de saúde, focando no nível individual do serviço prestado, ou seja, o atendimento realizado pelo profissional humano.

## O Repositório:
O repositório do projeto QualiCheck no GitHub conta com a documentação do projeto, contendo o registro das *sprints*, nesses registros são apresentados o *backlog* do produto, o PM Canvas e o mapa de empatia.
